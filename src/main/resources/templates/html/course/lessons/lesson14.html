MODYFIKACJA ZMIENNYCH

Każdej zmiennej, którą wykorzystywać będziesz w swoim kodzie, możesz modyfikować wartość. W jaki sposób? Nie jest to trudne, po prostu wpisujesz nazwę tej zmiennej a następnie przypisujesz jej zmienioną wartość. Istnieje również możliwość przypisania jednej zmiennej wartości innej zmiennej. Popatrz na kod:

#include <iostream>

#include <iostream>

using namespace std;


int main()

{

int a(4), b(2); // Deklaracja dwóch zmiennych.


cout << "Wartosc zmiennej a: " << a << ", wartosc zmiennej b: " << b << endl;


a = 10; // Zmiana wartości zmiennej a

b = 7; // Zmiana wartości zmiennej b

cout << "Wartosc zmiennej a po zmianie: " << a << ", wartosc zmiennej b po zmianie: " << b << endl;


a = b; // Przypisanie zmiennej a wartości zmiennej b.

cout << "Wartosc zmiennej a: " << a << ", wartosc zmiennej b: " << b << endl;


return 0;

}

Przeanalizuj powyższy kod. Spróbuj, bez patrzenia na wynik przewidzieć, co pojawi się na konsoli.

Wynik:

Czy udało Ci się poprawnie przeanalizować kod? Jeśli tak, brawo! Jeśli nie spróbuj stworzyć nowy projekt, wykorzystać ten kod i modyfikować zmienne, tak, by za każdym razem otrzymać inny wynik.

Pamiętaj! w operacji przypisania zmianie ulega tylko wartość zmiennej znajdującej się po lewej stronie znaku =.



OPERACJE MATEMATYCZNE

Jakie znasz operacje matematyczne? Jest to na pewno dodawanie, odejmowanie, mnożenie, dzielenie. Każda z tych operacji ma swój operator. Dla dodawania jest to +, odejmowania -, mnożenia *, dzielenia /. Istnieje jeszcze jedna operacja matematyczna, której najprawdopodobniej nie znasz- dzielenie modulo- za pomocą tej operacji uzyskujemy resztę z dzielenia liczby, a jej operatorem jest %.

Jeśli w swoim programie potrzebujesz wykorzystywać operacje matematyczne na wartościach znajdujących się w Twoich zmiennych- nie ma problemu! Obliczenia wykonuje się w taki sam sposób, jakby to była liczba. Najlepiej przedstawić to zagadnienie na przykładzie:

#include <iostream>

using namespace std;


int main()

{

int a = 1;

int b = 3;

cout << "srednia liczby a i b: " << (a + b) / 2 << endl;


}

Wynikiem powyższego programu:

Co w przypadku, gdy wynik działania będzie nam potrzebny w dalszej części naszego programu? Wynik wybranego obliczenia, można również przypisać do zmiennych:

#include <iostream>

using namespace std;


int main()

{

int a = 1;

int b = 2;

int wynik = (a*b + 1) - 4; // przypisanie do zmiennej wynik, wyniku działani

// wyświetlenie wartości zmiennej wynik

cout << "wynik dzialania: "<<wynik << endl;


}

Jak widzisz, do zmiennej wynik została przypisana wartość, będąca wynikiem działania na zmiennych a i b. Wartość zmiennej wynik została wyświetlona na konsoli.

Na początku tego paragramu wymienialiśmy operacje matematyczne. Wśród nich była jedna, która najprawdopodobniej, jest dla Ciebie nowa. Jest to operacja modulo, która zwraca resztę z dzielenia dwóch liczb całkowitych( tylko i wyłącznie liczb całkowitych, gdybyś chciał wykonać tą operację na liczbach typu double- Twój program nie skompiluje się, wyrzucając błąd). Operację tą można zapisać w następujący sposób:

#include <iostream>

using namespace std;


int main()

{

int a = 10;

int b = 3;

int wynikModulo = a%b;

cout << "wynik dzielenia modulo: " << wynikModulo << endl;

}

Operacja modulo, którą zwraca liczbę 1, dlaczego? Dzielimy 10 na 3. Wynik tego dzielenia to 3 reszty 1, zatem zmienna wynikModulo będzie przechowywać wartość reszty-1.

WAŻNE!

  • Wykonując operacje matematyczne pamiętaj o kolejności wykonywania działań!

  • Dzieląc dwie zmienne typu int uzyskasz wynik typu int! Na przykład dla:

int a=10;

int b=4;

int wynik = a/b;

wynik to: 2


Omawiając zmienne matematyczne chcielibyśmy zwrócić twoją uwagę na to, że oprócz zmieniania wartości danej zmiennej, którego nauczyłeś się w poprzedniej lekcji, możemy również wykonać operacje matematyczne na tej zmiennej, czyli np. zwiększyć jej wartość o 4, podzielić jej wartość na 4 itp.

int zmienna=10;

zmienna=zmienna+10; // wynik operacji- zwiększenia wartości o 10, zapisujemy w zmiennej, na której tą operację wykonaliśmy.


INKREMENTACJA I DEINKREMENTACJA

Właśnie poznasz nowe operatory, o których zapewnie jeszcze nigdy nie słyszałeś. Gdy chcesz zwiększyć bądź zmniejszyć wartość zmiennej o 1, możesz to zrobić za pomocą specjalnych operatorów: ++ oraz --.

Dla dodawania:

zmienna++; //post-inkrementacja

++zmienna; //pre-inkrementacja

Inkrementacja to proces zwiększania wartości zmiennej o 1. Oba zapisy, które widzisz powyżej, są równoznaczne z : zmienna=zmienna+1 ; O różnicy post i pre-inkrementacji powiemy Ci w dalszych modułach. Póki co, przyjmij, że oba te zapisy są równoznaczne.

Dla odejmowania:

zmienna--; //post-dekrementacja

--zmienna; //pre-dekrementacja

Skoro inkrementacją nazywamy proces zwiększania wartości zmiennej o 1, to dekrementacją nazwiemy proces zmniejszania zmiennej o 1. Oba te zapisy są równoznaczne z: zmienna=zmienna-1;

Popatrz na przykład:

int main()

{

int a = 5;

int b = 2;

int wynik = a + b * 3;

a = a + 2;

b = b - 1;

int wynik2 = a + b * 3;

cout << "wynik 1: " <<wynik << endl;

cout << "wynik 2: " <<wynik2 << endl;

}

Jeśli zmieniliśmy wartość zmiennej, to zmienna wynik wykona operacje matematyczne ale na zmienionych zmiennych, wynikiem powyższego programu będzie:

OPERACJE PORÓWNANIA

Kolejnymi operacjami, które chcemy omówić są operacje porównania. Operatory porównania, które wyróżniamy to:

Operator

Opis

>

... jest większe od ...

>=

... jest większe lub równe niż ...

<

... jest mniejsze od ...

<=

... jest mniejsze lub równe niż ...

==

... jest równe ...

!=

... jest różne od ...



Wynikiem wykonanej operacji porównania może być wartość true lub false, oznaczające odpowiednio prawdę i fałsz. Tak jak operacje matematyczne, operacje porównania mogą być wykonywane bezpośrednio na wartościach i zmiennych.

Na przykład:

#include <iostream>

using namespace std;

int main()

{

int a = 100 == 10;


cout << "a: " << a << endl;

cout << "b: " << (100 != 10) << endl;

cout << "c: " << (100 <= 10) << endl;

cout << "d: " << (100 >= 10) << endl;

cout << "e: " << (100 > 10) << endl;

cout << "f: " << (100 < 10) << endl;

}

Jak widzisz program zwrócił wyniki: true bądź false, w zależności od tego czy dane porównanie jest prawdziwe czy też nie. Do zmiennej a przypisana została wartość operacji porównania czyli false, a nie liczba.

Uwaga! Jeśli chcesz wykonywać operację porównania bezpośrednio przy wyświetlaniu, pamiętaj, że całe wyrażenie musisz zamknąć nawiasem. Bez tego- zwrócony zostanie błąd.

Uwaga pułapka! Odróżniaj operator przypisania = od porównania ==. Jeśli będziesz chciał porównać dwie wartości:

#include <iostream>

using namespace std;


int main()

{

int a = 3;

bool b= a = 5;

cout << "a = " <<a << endl;

cout << "b = " << b <<endl;

return 0;

}

Wynik jaki otrzymasz to:

Jak widzisz podczas inicjowania zmiennej b błędnie zainicjowano jej wartość, która miała być wynikiem porównania liczb 3 i 5 czyli: 0. Zamiast operacji porównania została wykorzystana operacja przypisania. W konsekwencji kompilator najpierw do zmiennej b przypisał wartość zmiennej i uznał to za prawdę ponieważ wartość tej zmiennej jest różna od zera. Dlatego za każdym razem, gdy wykonujesz porównanie równościowe, sprawdź, czy na pewno skorzystałeś z operatora porównania.

OPERACJE LOGICZNE

W języku C++ wyróżniamy trzy podstawowe operacje logiczne: iloczyn logiczny (AND / i ), sumę logiczną (OR/ lub) oraz negację. Operatory logiczne dla tych 3 operacji to:

Nazwa

Język C++

Opis

AND/i

&&

Iloczyn logiczny - wszystkie wartości muszą być prawdziwe, aby została zwrócona prawda.

OR/lub

||

Suma logiczna - co najmniej jedna z wartości musi być prawdziwa, aby została zwrócona prawda.

negacja

!

Zanegowanie wartości - czyli zwrócenie wartości przeciwnej.



Omówmy poniższy przykład, na którym najprościej będzie Ci zrozumieć w jaki sposób działają wymienione przez nas operacje logiczne:

int main()

{

bool a = true;

bool b = true;

bool c = false;

bool d = false;


cout << "NEGACJA: " << endl;

cout << !a << endl;


cout << "ILOCZYN LOGICZNY: " << endl;

cout << (a && b) << endl; // prawda i prawda

cout << (a && c) << endl; // prawda i fałsz

cout << (c && b) << endl; // fałsz i prawda

cout << (c && d) << endl; // fałsz i fałsz


cout << "SUMA LOGICZNA: " << endl;

cout << (a || b) << endl; // prawda lub prawda

cout << (a || c) << endl; // prawda lub fałsz

cout << (c || b) << endl; // fałsz lub prawda

cout << (c || d) << endl; // fałsz lub fałsz

}

Wyniki, które otrzymujemy są odzwierciedleniem funkcji jakie dane operacje logiczne posiadają. W przypadku negacji wartości a, która jest prawdziwa uzyskaliśmy wartość przeciwną czyli fałsz. Dla iloczynu logicznego wartość true uzyskaliśmy tylko i wyłącznie dla pierwszego przypadku, gdy obie wartości były prawdą. Dla sumy logicznej tylko i wyłącznie dla przypadku, gdy obie wartości są fałszywe uzyskujemy fałsz.

OPERACJE LOGICZNE Z OPERACJAMI PORÓWNANIA

Omówiliśmy już podstawowe operacje matematyczne, logiczne oraz porównania. Znasz już podstawy, jeśli jeszcze nie do końca jesteś przekonany, że zrozumiałeś dany materiał, powróć do tych zagadnień. Jeśli czujesz, że "wiesz o co biega", przejdźmy do bardziej skomplikowanego zagadnienia tzn. połączmy operacje porównania z logicznymi.

Popatrz na poniższy przykład:

#include <iostream>

using namespace std;


int main()

{

int wartosc = 3;

bool wynik = wartosc > 10 || wartosc < 100 && wartosc != 10;


cout << "wynik: " << wynik << endl;

}

Zmienna wynik przyjmuje wynik operacji logicznych, wewnątrz których wykonywane są operacje porównania. Kolejność wykonywania operacji logicznych jest następująca:

Najważniejszy jest operator negacji, później wykonywane są operacje iloczynu logicznego, a na samym końcu sumy logicznej.Jeśli operacje mają ten sam priorytet- idziemy od lewej strony.

Zatem jaki wynik powinniśmy otrzymać w powyższym programie? Na początku wykona się operacja iloczynu logicznego: 3 < 100 zwraca prawdę 3!=10 również zwraca prawdę czli mamy: prawda && prawda ---> prawda. Następnie otrzymany wynik będzie brał udział w operacji sumy logicznej: 3>10 to fałsz, fałsz || prawda ----> prawda. Zaten na konsoli powinno się ukazać wynik:1, czy tak jest? Sprawdźmy!
