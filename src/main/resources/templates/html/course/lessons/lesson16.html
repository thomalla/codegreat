LEKCJA 6: POBIERANIE WARTOŚCI OD UŻYTKOWNIKA



Dzięki naszemu kursowi poznałeś już instrukcję cout, która pozwala na wyprowadzenie danych na zewnątrz programu i wyświetlenie ich w konsoli. Skoro istnieje taka instrukcja, oczywistym jest, że język udostępnia również instrukcję umożliwiającą wprowadzenie danych do programu. Taką instrukcją jest instrukcja cin. Skoro dane są wprowadzane do programu, zatem kierunek nawiasów jest również przeciwny niż w przypadku instrukcji cout. Najlepiej zaprezentować to zagadnienie na przykładzie:

#include <iostream>

#include <string>

using namespace std;


int main()

{

string imie =" "; // Przygotowujemy komórkę pamięci na imię użytkownika


cout << "Jak masz na imie?" << endl;

cin >> imie; // W przygotowanej komórce chcemy zapisać imię użytkownika


cout << "Witaj " << imie << " !" << endl; //wyświetlamy komunikat zawierający imię użytkownika;

}



Na początku na konsoli pojawia się pytanie o imię, następnie w następnej linii pojawia się migający kursor, oznacza to, że komputer oczekuje, że właśnie w tym miejscu użytkownik coś napisze. Zauważ, że przygotowałeś już komórkę pamięci na wpisane przez Ciebie imię, zatem wpisana przez Ciebie wartość powędruje właśnie do tej komórki pamięci, zastępując jej obecną wartość( w tym przypadku pusty łańcuch, ale może być to np 0 dla zmiennej int). Na koniec, wyświetlamy wartość naszej zmiennej imię.



Przeróbmy nasz program tak, by użytkownik podawał osobno imię i nazwisko, a następnie wyświetlmy te dane. Zachęcamy Cię, byś spróbował napisać ten program sam, dopiero później sprawdź poniższy przykład:



#include <iostream>

#include <string>

using namespace std;


int main()

{

string imie = " "; // Przygotowujemy komórkę pamięci na imię użytkownika

string nazwisko = ""; // Przygotowujemy komórkę pamięci na nazwisko uzytkownika


cout << "Jak masz na imie?" << endl;

cin >> imie; // W przygotowanej komórce chcemy zapisać imię użytkownika


cout << "Jak masz na nazwisko? " << endl;

cin >> nazwisko; // W przygotowanej komórce chcemy zapisać nazwisko użytkownika

cout << "Witaj " << imie << " " << nazwisko<< " !" << endl; //wyświetlamy komunikat zawierający imię i nazwisko użytkownika;

}





Jak widzisz na konsoli wyświetliły się obie wartości zmiennych. Tak jak już wspomnieliśmy możesz tak działać na każdym typie zmiennych, nie tylko dla zmiennych łańcuchowych np:

#include <iostream>

#include <string>

using namespace std;


int main()

{

string imie = " ";

string nazwisko = "";

int wiek(0);

float wzrost(0.0);

cout << "Jak masz na imie?" << endl;

cin >> imie;


cout << "Jak masz na nazwisko? " << endl;

cin >> nazwisko;


cout << "Ile masz lat? " << endl;

cin >> wiek;


cout << "Ile masz m wzrostu? " << endl;

cin >> wzrost;

cout << "Witaj " << imie << " " << nazwisko << " !" << endl;

cout << "Masz: " << wiek << " lat, i " << wzrost << " m wzrostu" << endl;

}



Co w przypadku, gdy chcielibyśmy, żeby użytkownik w jednej linii wpisywał imię i nazwisko i było ono zapisywane w jednej zmiennej? Niestety pojawia się problem związany ze spacją. Gdy wpiszemy imię i nazwisko, to poprzez instukcję cin, do naszej zmiennej zapisze się tylko imię, a wszystkie dane po spacji, zostaną pominięte. Nie ma sytuacji bez wyjścia, C++ udostępnia rozwiązanie tej sytuacji. Doskonale do tego celu nadaje się funkcja getline(), która pobierze cały wiersz i zapisze go w danej zmiennej:

#include <iostream>

#include <string>


using namespace std;


int main()

{

string imieNazwisko(" ");

cout << "Jak sie nazywasz? " << endl;

getline(cin, imieNazwisko);

cout << "Witaj, " << imieNazwisko<< "!" << endl;

}



Pamiętaj! Kolejność w funkcji getline() jest zawsze taka sama tj. na początku cin, a później zmienna do której zapisujemy dane wpisane przez użytkownika.



Uwaga! Popatrz na poniższy przykład:

int main()

{

string imieNazwisko(" ");

int wiek(0);

cout << "Ile masz lat" << endl;

cin >> wiek;

cout << "Jak sie nazywasz? " << endl;

getline(cin, imieNazwisko);

cout << "Witaj, " << imieNazwisko<< "!" << endl;

}



Wynik działania tego programu:



Jak widzisz, coś poszło nie tak, nie pojawiło się pole, w którym mogliśmy wpisać imię i nazwisko. Problem ten można rozwiązać poprzez wpisanie wiersza kodu cin.ignore(); po wierszu, w którym użyta została instrukcja cin:

#include <iostream>

#include <string>


using namespace std;


int main()

{

string imieNazwisko(" ");

int wiek(0);

cout << "Ile masz lat" << endl;

cin >> wiek;

cin.ignore();

cout << "Jak sie nazywasz? " << endl;

getline(cin, imieNazwisko);

cout << "Witaj, " << imieNazwisko<< "!" << endl;

cout << "Masz " << wiek << " lat" << endl;

}



Pamiętaj! Kiedy w jednym programie używane są zamiennie instrukcja cin i funkcja getline(), należy zawsze po instrukcji cin wpisać wiersz cin.ingore();















Dzięki tej lekcji:

  • wiesz w jaki sposób uzyskać zmienne wejściowe, wpisywanie przez użytkownika

  • znasz instrukcję cin i umiesz się nią posługiwać

  • znasz funkcję getline() i umiesz się nią posługiwać
