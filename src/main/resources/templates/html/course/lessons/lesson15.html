LEKCJA 5: STAŁE



W poprzednich lekcjach poznałeś zagadnienie operacji na zmiennych. Wiesz już w jaki sposób je modyfikować. Czy zastanawiałeś się może, czy są takie zmienne, których modyfikować nie można? Czy może niezależnie od rodzaju zmiennych, każdą zmienną można poddać modyfikacji? Śpieszymy się by odpowiedzieć Ci na te pytania. Otóż w C++ istnieją zmienne, których wartości po zainicjowaniu nie możemy zmieniać, tymi zmiennymi są stałe. W momencie tworzenia stałej, musimy jej nadać wartość początkową( zainicjalizować) i nie mamy możliwości by tą wartość, w trakcie działania programu, zmieniać. Stałe mogą być zmiennymi każdego z typów udostępnianych przez C++, może to być zarówno liczba, jak i napis, czy znak.

DEFINIOWANIE STAŁYCH:

Stałe w C++ możemy deklarować na 2 sposoby:

  1. Za pomocą dyrektywy #define ( przestarzała wersja), brak konieczności podawania typu zmiennej.

  2. Za pomocą słowa kluczowego const



#include <iostream>

using namespace std;


#define stala1 12 // pierwszy sposób, pamiętaj, że instrukcja nie kończy się ;


int main()

{

// drugi sposób, oba sposoby są poprawne

const int stala2 = 12;

int const stala3 (12);


cout << stala1 << " " << stala2 << " " << stala3 << " " << endl;

}


Kiedy moglibyśmy użyć stałych? Kiedy definiujemy jakąś stałą matematyczną np. wartość przyspieszenia ziemskiego, kiedy mamy pewność, że dana zmienna przez cały czas trwania programu nie będzie ulegała zmianie. Korzystanie ze stałych umożliwia kompilatorowi wygenerowanie bardziej wydajnego kodu, dlatego jeśli masz możliwość to używaj stałych w swoich programach.
